# 캐릭터 스텟 한글판

## 스킬창.jpg 설명
image/스킬창.jpg
인게임에서 대부분 K를 누르면 열 수 있는 스텟창이다.
1레벨업 할 때마다 2포인트를 얻을 수 있다.
스킬 초기화권을 구매해서 스텟을 초기화 할 수 있다.

### 레이아웃
- 최상단: 직업, 레벨, 경험치, 포인트
- 좌측: 기본 스텟
- 우측: 최종 스텟 현황

### 레이아웃 디테일 설명
- 최상단: 내 정보가 있고, 우측 포인트는 현재 투자 가능한 포인트의 남은 수량
- 좌측: STR, DEX, INT, DEF
* STR - 데미지 상승
* DEX - 이동속도, 공격속도
* INT - 추가 데미지
* DEF - 방어력
- 우측: 현재 내 캐릭터가 가지는 스텟의 상세 내용
* 전투: 공격력, 방어무기데미지, 최종 데미지, 추가 데미지, 치명확률, 치명타 데미지, 쿨타임 감소
* 서바이벌: 방어력, 최대 체력, 체력 재생
* 기동성: 이동 속도

### 주문
- image를 참고해서 레이아웃을 비슷하게 하고, 그냥 한글화처럼 표시해주면 좋겠어. 
- 구현될 때 색상과 위치도 비슷하게 만들어줘.
- 레시피랑 메뉴를 다르게 해서 레시피는 레시피만 보이게, 스텟 한글화 누르면 스텟 한글화된(css와 js를 써서 더 예쁘게) 모양이 보이길 원해.
- 추가로 레시피와 스텟 등 사이트 자체를 모바일에서 최적화해서 보이게 해줘. 모바일 UI,UX가 세로로 길게 늘어지니까 유저 프랜들리하지 못해서 인터넷 쇼핑몰처럼 사이트처럼 만들어줘.
